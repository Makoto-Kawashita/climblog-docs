# ClimbLog - 位置情報の許可設定

## 画面の概要

このダイアログは、ClimbLogアプリが位置情報へのアクセス許可を求める画面です。天気情報を自動で取得・入力するために、現在地の使用許可が必要となります。

## 主要な機能・UI要素の説明

- **許可リクエストメッセージ**: 「"ClimbLog"に位置情報の使用を許可しますか?」という確認メッセージ
- **使用目的の説明**: 「天気を自動入力するため、現在地を使用します。」という利用目的の記載
- **選択ボタン（3つのオプション）**:
  - **1度だけ許可**: 今回のみ位置情報の使用を許可
  - **アプリの使用中は許可**: アプリを開いている間は位置情報の使用を許可
  - **許可しない**: 位置情報の使用を拒否

## 基本的な操作方法

1. アプリを初めて起動した際、または位置情報が必要な機能を使用する際にこのダイアログが表示されます
2. 3つの選択肢から希望するオプションをタップして選択してください
3. 選択後、自動的にアプリのメイン画面に進みます

## ヒントや注意事項

- **推奨設定**: 天気情報の自動入力機能を活用したい場合は「**アプリの使用中は許可**」を選択することをおすすめします
- 「許可しない」を選択した場合でも、後からiPhoneの「設定」→「ClimbLog」→「位置情報」から変更可能です
- 位置情報は天気の自動取得にのみ使用され、クライミング記録と合わせて当日の気象条件を記録できます
- バッテリー消費を抑えたい場合は「1度だけ許可」を選び、必要な時だけ許可する方法もあります
