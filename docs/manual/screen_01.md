# 位置情報の使用許可

## 画面の概要
ClimbLogアプリが位置情報へのアクセス許可を求めるシステムダイアログです。天気情報を自動入力するために、現在地の取得が必要となります。

## 主要なUI要素の説明

- **ダイアログタイトル**: 「"ClimbLog"に位置情報の使用を許可しますか?」
- **説明文**: 「日本語: 天気を自動入力するため、現在地を使用します。」
- **選択ボタン**:
  - **1度だけ許可**: 今回のみ位置情報の使用を許可します
  - **アプリの使用中は許可**: アプリを開いている間は常に位置情報を使用できます
  - **許可しない**: 位置情報へのアクセスを拒否します

## 基本的な操作方法

1. ダイアログが表示されたら、3つの選択肢から希望するオプションをタップします
2. **推奨**: 「アプリの使用中は許可」を選択すると、クライミング記録時に毎回天気情報が自動で入力されます
3. 選択後、自動的にアプリのメイン画面に戻ります

## ヒントや注意事項

> **💡 ヒント**
> - 「アプリの使用中は許可」を選ぶと、登った日の天気を自動記録でき、後から振り返る際に便利です
> - 位置情報はあくまで天気取得のために使用され、クライミングジムや岩場の場所特定にも活用されます

> **⚠️ 注意**
> - 「許可しない」を選択した場合、天気情報は手動で入力する必要があります
> - 後から設定を変更したい場合は、iPhoneの「設定」→「ClimbLog」→「位置情報」から変更できます
